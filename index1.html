Penjelasan

Depresiasi / th = (P – S) / N
  dimana
        P = Nilai Investasi
        S = Nilai Sisa
        N = Masa Pakai
Contoh :
      P = 900; N = 5; S = 70
Depresiasi / th = (P – S) / N
                                = (900 – 70) / 5
                                = 166
